# crypto-banking-system

A C++ based console application that simulates a basic cryptocurrency banking system with user authentication, account management, and cryptocurrency trading capabilities.

## Features

- **User Authentication**
  - Secure login system
  - New user registration
  - Session management

- **Banking Operations**
  - Deposit money
  - Withdraw funds
  - View account balance
  - Transaction history

- **Cryptocurrency Trading**
  - Buy/Sell Ethereum and Bitcoin
  - Track cryptocurrency holdings
  - Simulated market fluctuations

- **Data Persistence**
  - Save account information
  - Store transaction history
  - User credentials storage

## Prerequisites

- C++ Compiler (supporting C++11 or later)
- Standard Template Library (STL)
- Basic file I/O permissions
